# 🚀 Galaxy Defender

우주를 구하는 액션 슈팅 게임입니다!

## 📖 게임 소개

Galaxy Defender는 HTML5 Canvas와 JavaScript로 제작된 우주 슈팅 게임입니다. 적들을 처치하고 보스를 물리치며 최고 점수를 달성하세요!

## 🎮 게임 플레이

### 조작 방법
- **키보드**: 방향키 또는 WASD로 이동, 스페이스바로 발사
- **터치**: 모바일에서 화면을 터치하여 이동, 자동 발사
- **일시정지**: ESC 키 또는 다른 탭으로 이동 시 자동 일시정지

### 게임 목표
- 적들을 처치하여 점수를 획득하세요
- 코인을 수집하여 업그레이드를 구매하세요
- 웨이브를 클리어하고 보스를 물리치세요
- 최고 점수를 달성하세요!

## ✨ 주요 기능

### 게임 시스템
- **웨이브 시스템**: 난이도가 점진적으로 증가하는 웨이브 진행
- **보스전**: 일정 점수 달성 시 강력한 보스 등장
- **콤보 시스템**: 연속 처치 시 콤보 보너스 점수
- **무적 시간**: 피격 후 일정 시간 무적 상태

### 파워업 아이템
- **쉴드**: 한 번의 피해를 막아주는 방어막
- **트리플 샷**: 3발 동시 발사
- **강화 탄환**: 더 강력한 탄환
- **코인**: 업그레이드 구매에 사용

### 특수 공격
- **미사일**: 넓은 범위의 폭발 공격
- **레이저**: 관통하는 강력한 레이저 빔
- **스톰**: 화면 전체를 공격하는 폭풍

### 업그레이드 상점
- 플레이어 속도 증가
- 최대 생명력 증가
- 탄환 데미지 증가
- 특수 공격 충전량 증가

### 설정
- 배경음악 ON/OFF
- 효과음 ON/OFF
- 다크모드 지원

## 🛠️ 기술 스택

- **HTML5 Canvas**: 게임 렌더링
- **JavaScript (Vanilla)**: 게임 로직
- **Web Audio API**: 사운드 시스템
- **LocalStorage**: 점수 및 업그레이드 저장

## 📁 프로젝트 구조

```
Galaxy_Defender/
├── index.html          # 메인 HTML 파일
├── css/
│   └── style.css      # 게임 스타일
├── js/
│   ├── main_screen.js      # 게임 상태 관리 및 메인 루프

│   ├── game_objects.js     # 게임 오브젝트 정의

│   ├── game_ui.js         # UI 관리
│   ├── game_storage.js    # 로컬 스토리지 관리
│   ├── game_audio.js      # 오디오 시스템
│   └── option_screen.js   # 설정 화면
├── images/

└── README.md
```

## 🚀 실행 방법

1. 저장소를 클론하거나 다운로드합니다:
```bash
git clone https://github.com/Socoms/Galaxy_Defender.git
```

2. `index.html` 파일을 웹 브라우저에서 엽니다.

3. 게임을 즐기세요!

**참고**: 최신 버전의 웹 브라우저(Chrome, Firefox, Edge, Safari)를 사용하세요.

## 🎯 게임 팁

- 적들의 패턴을 파악하고 예측하여 회피하세요
- 코인을 수집하여 업그레이드를 구매하면 생존률이 높아집니다
- 특수 공격은 보스전에서 유용합니다
- 콤보를 유지하면 보너스 점수를 얻을 수 있습니다
- 일시정지 기능을 활용하여 전략을 세우세요
